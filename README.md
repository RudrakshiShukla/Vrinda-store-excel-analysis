# Vrinda-store-excel-analysis

 ##  Project Overview
This project analyzes the sales performance of Vrinda Store using Microsoft Excel.
The objective is to understand customer behavior, sales trends, and revenue contribution across different channels, categories, and regions.

---

## Objective
- Analyze overall sales and order trends  
- Identify top-performing products and categories  
- Understand customer demographics and purchase patterns  
- Support business decisions using data insights  

---

## Tools & Techniques Used
- Microsoft Excel  
- Pivot Tables  
- Charts & Graphs  
- Data Cleaning & Formatting  
- Basic Excel Formulas  

---

##  Dataset
The dataset used in this project is included in this repository:
- *Vrinda_Store_Data.xlsx*

The dataset contains order details such as order ID, date, customer gender, age group, product category, sales amount, and sales channel.

---

## Key Insights
- Women customers contribute more to overall sales than men  
- Maximum sales come from the adult age group  
- Amazon, Flipkart, and Myntra are the top sales channels  
- Certain product categories generate the highest revenue
